Questions

You might be wondering

Is my chat private?+
Yes, completely. Your .txt file is analysed entirely in your browser — it is never uploaded to any server. Only your names, message count, and date range are used to make the cards.
How do I export my WhatsApp chat?+
Android: Open chat → ⋮ → More → Export chat → Without Media → save the .txt file.

iPhone: Open chat → Tap the contact name → Export Chat → Without Media. The file will be named "WhatsApp Chat with [Name].txt".
How long does generation take?+
About 20–40 seconds for all 9 cards. Everything runs in your browser — you'll see a live progress bar and each card appear as it renders.
What format are the cards?+
Each card is a 1080×1350 JPEG — the perfect size for Instagram carousels. They're bundled in a ZIP for easy download. The Keepsake package generates 2× size (2160×2700), ideal for printing and framing.
Can I make this for a friend?+
Absolutely — it makes a beautiful gift. Ask your friend to export the chat and share the .txt file with you, then fill in their names and upload a photo of them.
